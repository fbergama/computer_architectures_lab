# Computer Architecures Laboratory

This repository contains examples and materials for the [Computer Architectures Laboratory](https://www.unive.it/data/insegnamento/575972) course, Bachelor's degree in computer science, [Ca' Foscari University of Venice](https://www.unive.it).

Examples are meant to be executed in the ARM64 virtual machine (see the official Moodle page)
